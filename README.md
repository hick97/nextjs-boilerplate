# Commands

### Install

```sh
yarn install
```

### Build

```sh
yarn build
```

### Usage

```sh
yarn run start
```

### Development mode

```sh
yarn run dev
```

### Running storybook

```sh
yarn storybook
```

### Running lint

```sh
yarn lint
```

### Running tests

```sh
yarn test
or
yarn test:watch (Watch mode)
```

### Running component generator

```sh
yarn generate
or
yarn generate YourComponentName
```
